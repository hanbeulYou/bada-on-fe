# 바다온(BadaOn)

## 소개 🌊

바다ON은 제주도의 실시간 해양 환경 정보를 제공하는 서비스입니다. 스노클링, 서핑 등 해양 활동을 즐기는 사용자들에게 맞춤형 정보와 안전 가이드를 제공합니다.

## 주요 기능 ⭐️

1. 실시간 해양 환경 정보

- 기상 조건 (기온, 습도, 강수량 등)
- 해양 조건 (수온, 파고, 조석 등)
- 위치 기반 서비스

2. 활동별 맞춤 추천

- 각 해수욕장별 활동 추천 점수 제공
- 3일간의 예보 정보

3. 실시간 알림

- 경보 및 주의보 발생 시 실시간 위험 알림

## 기술 스택 🛠️

- React
- TypeScript
- React-Query
- Styled-Components
- Kakao Maps SDK, Kakao Map API
- React-Native

## API 및 데이터 출처

- 기상청 API
- 바다누리 API
